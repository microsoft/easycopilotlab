# 실습 ②: 에이전트 설정 변경
{: .no_toc }

| 시간 | 소요 | 수강생 역할 |
|:-----|:-----|:-----------|
| 11:20 | 10분 | 🟢 직접 실습 |

---

## Step 1 — 생성형 AI 설정 열기

1. Copilot Studio → HR 도우미 에이전트 열기
2. **설정 → 생성형 AI** 클릭

---

## Step 2 — 응답 길이 변경

1. **응답 길이** 옵션 확인 (짧게 / 보통 / 길게)
2. **"짧게"** 로 변경 → **저장**
3. 테스트 창에서 질문: `경비처리 어떻게 해?`
4. 답변 길이를 확인한 뒤, **"길게"** 로 변경 → **저장**
5. 같은 질문을 다시 입력하여 답변 길이 비교

{: .tip }
> 실무에서는 **"보통"** 이 대부분 적합합니다. "짧게"는 챗봇형, "길게"는 보고서형 에이전트에 적합합니다.

---

## Step 3 — 콘텐츠 조정 확인

1. **콘텐츠 조정** 옵션 확인
2. 부적절한 주제 차단 수준을 살펴봅니다
3. 기본값 그대로 유지 (실무에서는 조직 정책에 맞게 조정)

---

## Step 4 — 지식 없을 때 동작

1. **지식이 없을 때** 설정 확인
   - "AI가 일반 지식으로 답변" vs "모른다고 답변"
2. **"AI가 일반 지식으로 답변하도록 허용하지 않음"** 으로 변경 → **저장**
3. 테스트: `오늘 날씨 어때?`
   - 범위 밖 질문에 모른다고 거절하는지 확인

{: .important }
> 실무에서는 **"허용하지 않음"** 을 권장합니다. 지식 소스에 없는 내용을 AI가 지어내는(할루시네이션) 위험을 줄일 수 있습니다.

---

## 설정 변경 전후 비교 정리

| 설정 | 변경 전 | 변경 후 | 관찰 포인트 |
|:-----|:-----:|:-----:|:-----------|
| 응답 길이 | 보통 | 짧게 → 길게 | 같은 질문인데 답변 분량이 달라짐 |
| 지식 없을 때 | 허용 | 허용하지 않음 | 범위 밖 질문에 거절 메시지 |
