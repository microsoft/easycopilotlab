## 이 모듈에서 배우는 것

- Copilot Studio에서 에이전트를 **게시(Publish)**
- **M365 Copilot에서 에이전트 사용** — 몰입형(전용 화면) + @호출
- 다른 사람과 사용할 수 있도록 **공유(Share)**
- **재게시** 개념 이해

---

## 용어 먼저 정리

| 용어 | 뜻 |
|:-----|:---|
| **게시(Publish)** | Copilot Studio의 최신 변경사항을 사용 채널에 반영하는 단계 |
| **채널(Channel)** | 에이전트를 **어디서** 사용할지 정하는 설정 (Teams, M365 Copilot, 웹사이트 등) |
| **공유(Share)** | 특정 동료나 조직이 그 에이전트를 사용할 수 있게 권한을 주는 단계 |

---

## 게시 vs 채널 — 뭐가 다른가요?

| | 게시(Publish) | 채널(Channel) |
|:---|:---|:---|
| **하는 일** | 에이전트의 최신 버전을 "배포" | 에이전트를 사용할 **위치(경로)**를 연결 |
| **비유** | 앱을 앱스토어에 **업로드** | 앱스토어에서 **어떤 기기에 설치**할지 선택 |
| **횟수** | 수정할 때**마다** 반복 | 처음 한 번만 설정하면 유지 |
| **안 하면?** | 변경 사항이 반영되지 않음 | 게시해도 사용할 수 있는 곳이 없음 |

{: .highlight }
> 정리: **채널**은 "어디서 쓸지" → 처음 한 번 설정  
> **게시**는 "최신 버전 반영" → 수정할 때마다 반복

---

## 언제 게시해야 하나요?

| 상황 | 게시 필요? |
|:----|:----------|
| 에이전트를 **처음 만들고** 사용하려 할 때 | ✅ 필수 |
| 지침(Instructions)을 **수정**했을 때 | ✅ 필수 |
| 참조 자료(Knowledge)를 **추가·삭제**했을 때 | ✅ 필수 |
| 토픽(Topic)을 **새로 만들거나 변경**했을 때 | ✅ 필수 |
| 커넥터·흐름을 **연결**했을 때 (M11~M12) | ✅ 필수 |
| 테스트 패널에서 **미리보기만** 할 때 | ❌ 불필요 |

{: .warning }
> 게시하지 않으면 **테스트 패널에서는 보이지만**, Copilot·Teams 등 실제 채널에서는 이전 버전이 그대로 유지됩니다.

## 사용 채널 4가지

| 채널 | 추천 상황 | 오늘 실습 |
|:-----|:---------|:---------|
| **M365 Copilot (몰입형)** | 에이전트 전용 화면에서 집중 대화 | ✅ 실습 |
| **M365 Copilot (@호출)** | 빠르게 사용 (인컨텍스트) | ✅ 실습 |
| Microsoft Teams | 팀 내부용·협업 도구 연동 | 소개만 |
| 웹사이트 임베드 | 외부 고객 대상 | 소개만 |

## 게시 후 공유

에이전트를 동료와 공유하려면:
- 에이전트 **설정 → 공유** → 동료 이메일 입력
- 또는 관리자가 **조직 전체 사용 정책**으로 노출

---

## 재게시

{: .important }
> **게시는 1회성이 아닙니다.** 에이전트를 수정할 때마다 다시 게시해야 Copilot에 반영됩니다.

앞으로 M11~M16에서 커넥터·흐름·MCP 등을 추가할 때마다 → **다시 게시 버튼** 누르기  
→ Copilot의 에이전트에 새 기능이 즉시 반영됩니다.

{: .tip }
> 채널은 다시 설정할 필요 없습니다. **게시만 다시 누르면** 연결된 모든 채널에 자동 반영됩니다.

---

## 핵심 정리

1. **채널** = 에이전트를 어디서 사용할지 (처음 한 번 설정)
2. **게시** = 최신 버전을 연결된 채널에 반영 (수정 시마다 반복)
3. **몰입형** = Copilot에서 에이전트 이름 클릭 → 전용 화면에서 집중 대화
4. **@호출** = Copilot 채팅에서 빠르게 질문
4. **공유** = 다른 사람이 쓸 수 있도록 권한을 주는 단계
5. **수정 후에는 반드시 재게시**

## FAQ

| 질문 | 답변 |
|:-----|:-----|
| Copilot에서 에이전트가 안 보여요 | 게시 후 1~3분 소요됩니다. 새로고침 후 다시 확인해 보세요. |
| @호출이 안 됩니다 | 게시 여부 확인, Copilot 채널 활성화 확인, 관리자 정책 확인이 필요합니다. |
| 다른 팀원도 바로 쓸 수 있나요? | 공유 설정에서 추가하거나 관리자 정책으로 조직 전체에 노출할 수 있습니다. |
| 게시를 되돌릴 수 있나요? | 채널에서 비활성화하거나 에이전트를 비게시 상태로 전환할 수 있습니다. |
