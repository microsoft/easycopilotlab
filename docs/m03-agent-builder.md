## 에이전트 빌더란?

M365 Copilot 안에 내장된 **간편 에이전트 생성 도구**입니다.  
자연어로 설명하면 30초 만에 에이전트가 만들어집니다.

{: .highlight }
> 스마트폰 카메라처럼 빠르고 간편합니다. 나중에 배울 Copilot Studio는 DSLR처럼 정밀합니다.

## 실습: 첫 에이전트 만들기

### Step 1 — 에이전트 빌더 접속
1. [M365 Copilot](https://copilot.microsoft.com) 또는 Teams Copilot 채팅 접속
2. **에이전트 빌더** 선택

### Step 2 — 자연어로 설명 입력
입력 예시:
```
한국어로 친절하게 응대하는 사내 FAQ 도우미를 만들어줘.
복리후생, 연차, 경비처리 질문에 답변하는 에이전트야.
```

### Step 3 — 자동 생성 확인
에이전트 빌더가 자동으로 생성합니다:
- **이름** — 예: "사내 FAQ 도우미"
- **지침(Instructions)** — 역할과 행동 방식
- **설명** — 한 줄 요약

### Step 4 — 테스트
테스트 패널에서 질문을 입력해 보세요:
- "연차 며칠이야?"
- "경비처리 어떻게 해?"

### Step 5 — 지침 수정 후 재테스트
지침을 한 줄 수정해 보세요. 예: "존칭" → "반말"  
→ 에이전트의 답변 톤이 즉시 바뀌는 것을 확인!

{: .tip }
> **텍스트 한 줄의 변경**이 에이전트 성격을 완전히 바꿉니다. 이것이 "문과생이 더 잘하는 이유"입니다.

## 6가지 샘플 체험

강사가 제공하는 지침을 복사·붙여넣기하면 바로 사용할 수 있습니다.  
관심 있는 샘플을 골라 직접 만들어 보세요.

| 샘플 | 핵심 기능 | 추천 대상 | 난이도 |
|:-----|:---------|:---------|:------|
| **A. 역할극 트레이너** | 캐릭터 몰입 + 피드백 | 영업/CS/협상 업무 | ⭐ |
| **B. 글쓰기 스타일 교정** | 문장 → 4가지 스타일 변환 | 보고서·이메일 작성자 | ⭐ |
| **C. 전문가 토론회** | 4명 페르소나 토론 진행 | 다양한 시각이 필요할 때 | ⭐⭐ |
| **D. 회의록 정리** | TXT/Word → 자동 정리 | 회의가 많은 분 | ⭐ |
| **E. 영수증 경비처리** | 이미지/PDF → 경비 추출 | 출장·경비 처리 빈번 | ⭐⭐ |
| **F. 발표자료 코치** | PPT/PDF → 피드백 + 개선안 | 발표자료 작성자 | ⭐⭐ |

{: .note }
> 6가지 모두 해볼 필요 없습니다. **1~2개만** 골라 직접 체험하세요.

## 에이전트 빌더 vs Copilot Studio

| 항목 | 에이전트 빌더 (스마트폰) | Copilot Studio (DSLR) |
|:-----|:---------------------|:---------------------|
| 생성 속도 | ⚡ 30초 | 🔧 설정 필요 |
| 지침 | ✅ 기본 가능 | ✅ 더 세밀하게 |
| 지식 연결 | 파일 업로드 기본 | 다양한 소스 연결 |
| Topic/Flow | ❌ 불가 | ✅ 대본 + 손발 |
| 배포 채널 | 제한적 | Teams·웹·앱 전체 |

{: .highlight }
> 오늘 오후부터는 **Copilot Studio(DSLR)**로 전환합니다. 에이전트 빌더에서 느낀 가능성을 Copilot Studio에서 본격적으로 구현합니다.

---

## 핵심 정리

1. 에이전트 빌더로 **30초 만에** 에이전트 생성 가능
2. **지침(텍스트)**을 바꾸면 성격이 즉시 변한다
3. 에이전트 빌더 = 스마트폰 카메라, Copilot Studio = DSLR
4. 오후에는 Copilot Studio에서 더 정교하게 만든다

## FAQ

| 질문 | 답변 |
|:-----|:-----|
| 에이전트 빌더로 만든 건 Copilot Studio로 옮길 수 있나요? | 네, Copilot Studio에서 확장·편집할 수 있습니다. |
| 회사 문서를 에이전트에 연결하려면? | 에이전트 빌더에서도 파일 업로드가 가능하지만, 더 정교한 연결은 Copilot Studio에서 합니다 (M6에서 실습). |
| 샘플 지침은 어디서 받나요? | 강사가 채팅으로 공유하거나 실습 가이드 문서에 포함되어 있습니다. |
| 영어로 지침을 써야 더 잘 되나요? | 한국어로 작성해도 잘 작동합니다. 한국어 지침 권장. |
