# 실습 ①: Excel 파일 준비
{: .no_toc }

| 시간 | 소요 | 수강생 역할 |
|:-----|:-----|:-----------|
| 15:20 | 5분 | 🟢 직접 실습 |

---

{: .important }
> **OneDrive for Business**와 **Excel Online (Business)** 접근 권한이 있어야 합니다. 조직 정책상 OneDrive 사용이 제한되어 있으면 같은 구조로 **SharePoint 문서 라이브러리**를 사용해도 됩니다.

1. **OneDrive** 접속 → 새 Excel 파일 생성: `대화기록.xlsx`
2. **Sheet1**에 테이블 만들기:

| 시간 | 사용자 | 질문 | 답변 |
|:-----|:------|:-----|:-----|
| (비워두기) | | | |

3. 표 전체 선택 → **"삽입" → "표"** → 확인
4. **저장**
