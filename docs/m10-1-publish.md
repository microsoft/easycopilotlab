# 실습 ①: 게시(Publish) & 채널 추가
{: .no_toc }

| 시간 | 소요 | 수강생 역할 |
|:-----|:-----|:-----------|
| 15:00 | 5분 | 🟢 직접 실습 |

---

## A. 게시(Publish)

1. Copilot Studio → 에이전트 편집 화면

   ![게시 버튼 위치](../assets/images/m10/image.png)

2. 우측 상단 **"게시(Publish)"** 클릭
3. 게시 확인 대화상자 → **"게시"** 클릭

   ![게시 확인 대화상자](../assets/images/m10/image2.png)

4. 게시 진행 중 화면 → **"닫기"** 클릭 (백그라운드에서 계속 진행됨)

   ![게시 진행 중](../assets/images/m10/image3.png)

{: .highlight }
> 게시 = 최신 변경사항을 **실사용 버전**에 반영하는 단계입니다. 게시하지 않으면 테스트 패널에서만 확인됩니다.

---

## B. 채널 추가 (Teams & Microsoft 365 Copilot)

게시가 완료되면 에이전트를 **어디서 사용할지** 채널을 연결해야 합니다.

1. 상단 탭에서 **"채널"** 클릭

   ![채널 탭 — 게시 진행 중](../assets/images/m10/image4.png)

2. 게시 완료 확인 — "에이전트 상태: 게시됨" 표시

   ![채널 탭 — 게시 완료](../assets/images/m10/image5.png)
